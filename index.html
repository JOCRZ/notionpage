Fatsort

üêß FATSort (Linux) Step-by-Step Guide

This process uses the command line to reorganize the file system's internal table (FAT), forcing the playback order to match the file names.

1. Prepare Your Files

  • Rename your files on your computer (not yet on the card) to have a numbered prefix that matches your desired playback order (e.g., 001.mp3, 002.mp3, 010.mp3, 100.mp3).
  • Copy all the prepared files and folders onto your memory card or USB drive.

2. Find the Device Path

  • Plug in your memory card.
  • Open a terminal and run the command to list block devices:
lsblk
  • Look for your memory card's size and note its device partition path, which will look like /dev/sdX1 or /dev/sdb1.

3. Unmount the Device

  • Crucially, you must unmount the partition before running fatsort. Replace /dev/sdX1 with your device path from Step 2:
sudo umount /dev/sdd1

If this fails, you may need to use the device's mount point (e.g., /media/user/MUSIC-CARD) instead.

4. Run FATSort

  • Execute the fatsort command on the unmounted partition. This rewrites the FAT table to match the file name order.
sudo fatsort -n /dev/sdd1

5. Safely Remove

  • After the command finishes, you can safely remove/eject the memory card and insert it into your MP3 player.

Yes, the fatsort utility offers several options beyond -n (Natural Order Sorting) to control the sorting of files and directories on a FAT filesystem.

OptionDescriptionExample
(Default)Alphanumeric (Standard alphabetical sort)sudo fatsort /dev/sdX1
-nNatural Order (The recommended option for media players). Sorts numerical sequences within filenames correctly (e.g., file3.mp3 before file21.mp3).sudo fatsort -n /dev/sdX1
-aASCIIbetical Order. Uses the raw ASCII values for sorting (may differ from your operating system's locale).sudo fatsort -a /dev/sdX1
-tSort by Time. Sorts files based on their last modification date and time.sudo fatsort -t /dev/sdX1
-rReverse Order. Reverses the result of the main sort (e.g., Z-A instead of A-Z). Use this with another sort option.sudo fatsort -nr /dev/sdX1 (Reverse Natural Order)
-RRandom Order. Sorts the entries in a random sequence.sudo fatsort -R /dev/sdX1

üóÇÔ∏è Directory/File Ordering Options

These options control whether directories are listed before or after files. They can be combined with any of the main sorting options above.

Option (-o)Description
-o dDirectories First (This is the default behavior).
-o fFiles First. Sorts all files before any directories.
-o aNo Differentiation. Files and directories are sorted together as a single list.

Example: To sort naturally, ignore case, and have files appear before directories:

Bash

sudo fatsort -n -c -o f /dev/sdX1
